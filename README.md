# Sistema-Notas

##  Descripción
Este proyecto se trata sobre el manejo del Try Catch (Excesiones). Aplicación de consola en C# para registrar 4 notas de un estudiante, calcular el promedio y determinar si aprueba o reprueba. Incluye validación de rango (0–100) y manejo de errores con try-catch-finally.




##  Tecnologías Utilizadas
C#
